Linguagens para Web

HTML (HyperText Markup Language)
É uma linguagem de marcação de hipertexto.
Possui tags que são os comandos utilizados pela linguagem.
CSS (Cascading Style Sheets)
CSS significa folhas de estilos em cascata.
É utilizada para formatar os elementos criados com HTML.
JS (JavaScript)
É uma linguagem de programação utilizada para injetar lógica em HTML.
É considerada a primeira extensão da linguagem HTML.
